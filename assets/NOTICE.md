# Asset attributions

## macOS folder bases (`folders/mac-os/`)

Re-encoded (to WebP) from the folder renders of
[FolderArt by christianvmm](https://github.com/christianvmm/folderart).
They reproduce the look of Apple macOS folder icons; Apple retains all rights to
its designs and trademarks. They are bundled here only to let users recreate the
native folder look for personal use. They are not covered by this project's MIT
license.

## Windows

No Windows folder asset is bundled. On Windows the base folder icon is read at
runtime from the OS itself (`imageres.dll.mun` / `shell32.dll`), so nothing is
redistributed.

## Linux

No Linux folder asset is bundled. On Linux the base folder icon is read at
runtime from the user's own current icon theme (e.g. Adwaita, Yaru, Breeze),
so nothing is redistributed.
